# 🍅 小番茄闹钟

一个基于 Electron 的桌面番茄钟应用，采用经典 25/5/15 番茄工作法。

## 功能

- **番茄计时** — 25 分钟专注 / 5 分钟短休息 / 15 分钟长休息
- **可视化进度环** — 不同模式有不同颜色（红/绿/蓝）
- **点击自定义时间** — 点击进度环可直接输入任意时长
- **暂停 / 恢复 / 重置 / 跳过**
- **统计面板** — 今日完成数、总计完成数、总专注分钟数
- **提示音 + 桌面通知**
- **托盘图标** — 最小化到系统托盘，右键菜单显示/隐藏
- **悬浮置顶** — 始终显示在其他窗口之上

## 安装

```bash
cd tomato-clock-app
npm install
```

## 运行

```bash
npm start
# 或
npx electron .
```

## 打包

```bash
npx electron-builder --win
```

生成的安装包在 `dist/` 目录下。

## 技术栈

- Electron 42
- 原生 HTML/CSS/JS
- Web Audio API 提示音
- SVG 进度环动画
